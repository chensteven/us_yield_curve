# us_yield_curve

# Setup


```bash
# use pyenv and virtualenv to create a Python virtual environment based on Python version 3.7.13 and the name of the virtual environment is us_yield_curve
pyenv virtualenv 3.7.13 us_yield_curve
# use pyenv to activate a virtual environment called us_yield_curve
pyenv activate us_yield_curve
# install the Python libraries that are specified in requirements.txt
pip install -r requirements.txt
```
